🔍 Flask Google Search API

Bu API Google araması yapar ve sonuçları JSON döndürür. Başlık/Açıklama çekilmez, yalnızca url, domain, hedef_site_mi alanları döner.

ParametreAçıklamaZorunlu
queryAranacak kelime/cümle✔
dilArama dili (örn: tr)✔
bolgeBölge kodu (örn: tr, us)✔
devicedesktop / mobile / tablet❌
